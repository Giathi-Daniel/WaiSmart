# Welcome to WaiSmart
WaiSmart is a React + TypeScript web app that showcases how smart water management systems can help rural communities monitor and optimize water usage. It features predictive insights, visual analytics, and user-friendly dashboards designed to promote sustainable irrigation practices.

### Tech Stack

* **Frontend:** React + TypeScript + Vite
* **UI:** Tailwind CSS + ShadCN UI
* **Visualization:** Sample data charts & simulated ML outputs

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Giathi-Daniel/WaiSmart.git
cd WaiSmart

# Install dependencies
npm install

# Start development server
npm run dev
```
